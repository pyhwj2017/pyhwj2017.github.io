濮阳市2017年黄河防汛物资供应调度保障预案

为提高防御黄河洪水灾害物资保障能力，明确保障责任，落实保障措施，最大程度地满足防御黄河洪水灾害物资保障需要，根据《中华人民共和国防洪法》、《中华人民共和国防汛条例》、《河南省黄河防汛物资管理细则》及《濮阳市黄河防洪预案》等规定，结合我市黄河防汛任务和各级洪水处理方案要求，特制定本预案。

一、黄河防汛物资的储备情况

我市黄河防汛储备物资由“国家储备、社会团体储备和群众储备”三部分组成，按照“常备不懈、保障急需”的原则，实施日常管护与组织供应。

（一）国家储备防汛物资是指黄河河务部门按照黄河防总储备定额和上级计划投资进行储备管理的黄河防汛专用物资，是我市黄河抗洪抢险先期投入使用的重要来源。该部分物资的购置、储存、管护等由黄河河务部门具体办理，濮阳黄河防汛专用物资和滞洪物资主要存储在濮阳第一黄河河务局、濮阳第二黄河河务局、范县黄河河务局、台前黄河河务局及渠村分洪闸管理处、张庄闸管理处的中心仓库和各河道工程应急仓库，濮阳临黄堤防、险工、控导工程、涵闸工程以及北金堤险工备有防汛（滞洪）石料。

黄河河务部门国家储备防汛物资又称国家常备防汛物资，其储备品种依据抢险常用而定，储备数量实行定额管理。储备的这部分物资主要用于黄河防汛抢险，接受国家防办、黄河防总办、省黄河防办应急调用；主要品种包括石料、铅丝（含铅丝笼网片）、麻绳、编织袋、土工布、发电机组、救生衣等，储备数量见下表。

（二）社会团体储备黄河防汛物资是我市黄河抗洪抢险物资保障的重要组成部分。这类储备是黄河抗洪非常时期，河务部门国家储备防汛物资因抢险消耗过大，储量严重不足时，为满足抢大险需要而采取的应急保障措施。储备的这部分物资优先用于黄河抗洪抢险救灾，若省内其它河流发生大的汛情或防汛突发事件，接受省、市防指跨地区应急调用。该部分物资实行定量管理，由沿黄各县防指按照防汛任务，结合抢险救灾需要向辖区内各行政机关，企、事业单位和社会团体下达储备任务，并于汛前储备到位。社会团体储备黄河防汛物资主要品种包括：各种抢险机械、交通运输车辆、通讯器材、救生器材、照明器材、铅丝、麻绳、篷布、编织袋、编织布、木材、钢材、水泥、沙石料、油料及生活用品等。

（三）群众储备黄河防汛物资是我市黄河抗洪抢险物资保障的有效组成部分，这些物资因部分来源单一，是国家和社会团体储备所不可替代的。群众备料由沿黄各县防指根据抗洪抢险需要进行安排，并向辖区内有关乡(镇)、村庄下达储备任务。群众储备黄河防汛物资主要包括各种抢险设备、救生器材、工具、交通运输车辆、树木及柳秸料等。

 二、防汛物资保障任务与措施

（一）保障任务

黄河防汛物资保障工作应从抗大洪、抢大险出发，结合防汛抢险、抗洪救灾需要，落实物资储备及保障措施，并确保其完好。根据黄河不同流量级洪水，不论黄河防洪工程出现何种险情，物资保障工作均要做到“处置及时、应对有力”，尽最大努力满足防汛抢险、抗洪救灾需要。

（二）保障措施

1. 加强领导，强化责任

物资保障工作实行行政首长负总责，有关单位或部门各司其职，工作人员责任明确、相互配合、协调一致的运行机制。要以政府保障为主线，一级保一级，层层抓落实。要落实好防汛物资供应、调度负责人和工作人员，并确保通讯联络畅通。要明确工作纪律，制定工作职责，建立请示报告制度和工作程序，确保紧急调用时保障有力。

2. 制定物资保障预案，落实汛前准备工作

汛前，市、县黄河防办要按照防洪任务，立足于“防大汛、抗大洪”，从实战出发，结合本地实际，制定物资保障预案。各级防汛物资储备单位，汛前要对所管理的防汛物资、抢险设备进行认真检查、维修、保养，确保完好、管用、配套。要结合物资保障工作内容，把保障任务分解到组、量化到人。要积极筹措物资，认真落实供货渠道，确保货源充足、型号管用，保证随调随有。要加强对抢险专用设备、器材操作人员进行岗位培训，达到能安全驾驶、熟练操作，能排除一般故障。

3. 筹划物资调运对策，夯实物资保障基础

各防汛物资保障单位，要以保障供应不误急需、反应快速为目标，从提高保障时效出发，对运输工具、运输路线、道路通行状况、物资装卸及运输安全等各方面工作进行充分预筹，制定有效措施，切实满足物资应急调运工作需要。运输防汛物资，做到“料源、运力、路线、人员、联络方式”五落实。在黄河主汛期，各物资保障部门要把组织物资调度列为一切工作的重中之重，认真筹措资源，实施有效调度，保障抗洪需要。公安、交通运输部门要确保运输干线畅通，在道路堵塞路段，要强化应急指挥机制，优先保证防汛物资运输车辆通行。

4. 加强信息沟通，保证决策准确

黄河主汛期，各级黄河防办要负责定期向上级黄河防办报告所管辖区内防汛物资储备情况、消耗动态，沟通可供物资资源信息，为上级黄河防办及时综合物资资源情况、科学组织调度提供依据。

三、防汛物资调度原则与程序

黄河防汛储备物资供应与调度按照“满足急需、先主后次、就近调运、先进先出”的原则办理。

（一）黄河河务部门储备的防汛物资，由沿黄黄河河务部门负责组织供应。黄河河务部门依据上级黄河防办或同级黄河防办调令，办理物资供应事项，组织物资到位。

（二）社会团体储备的防汛物资，由防汛物资储备责任单位负责组织供应。物资储备责任单位按照市防指以及所在地县防指调用物资意见，实施物资供应。

（三）群众备料防汛物资，由沿黄有储备任务的乡（镇）政府、村委会负责组织群众供应。有储备任务的乡（镇）政府或村委会按照所在县防指黄河防办下达的物资调令，负责组织落实到位。

（四）在黄河抗洪抢险紧急情况下，需要异地调度物资时，调度原则为：

1. 在濮阳市管辖范围内调度物资，由濮阳市黄河防办下达调令；需要跨市区调度物资时，由市黄河防办向省黄河防办提出物资调度申请，省黄河防办下达物资调令。

2. 调度物资一般遵循从上游市、县向下游市、县调，其运输工作由物资调出单位负责。

（五）我市防汛储备物资在满足不了黄河抗洪抢险使用时，统一由市防指黄河防办报告省防指黄河防办，或由市政府报告省政府请求支援。

（六）对外调用我市储备的黄河防汛物资时，各级黄河防办要做好物资调度的协调与组织工作，物资储备单位要按照上级调令落实到位。

（七）凡是执行“防汛物资调令”的交接双方，要按照防汛物资管理规定，认真履行物资交接手续，并将物资调度执行情况及时反馈上级防指黄河防办。物资接收方要安排专人加强对接收料物进行管理，确保料物不受损失。

四、各级洪水防汛物资保障方案

物资保障工作，贯彻“安全第一，常备不懈，以防为主，全力抢险”的防汛工作方针，遵照“一切为了抢险，一切服从抢险，一切服务于抢险”的原则，根据不同情况，采取有效措施，保证抗洪抢险工作顺利进行。

（一）当花园口站发生4000立方米每秒以下流量级洪水时，物资保障工作重点为新修建及根基不稳的靠河工程险情抢护。各县级黄河河务局要做好动用国家常备石料、麻绳、铅丝网片、木桩等防汛物资的准备工作，保证工程抢护使用。沿黄各乡（镇）、村群众做好为抗洪抢险运送柳秸料的准备工作，接指令后，按调用数量、时限要求保证组织运送到位。该流量级洪水如发生长历时下泄，要重点做好较大以上险情抢护期间的物资保障工作。

（二）当花园口站发生4000～6000立方米每秒流量级洪水时，我市辖区内河道工程全部靠河，随时有出险的可能，此时物资保障工作重点是险工和控导工程。保障方案为：

1. 黄河专业机动抢险队集结待命，做到设备完好、配套，人员到岗到位，能随时出动参加抗洪抢险。各县级黄河河务局做好动用国家储备防汛冲锋舟的准备工作，满足抗洪指挥、工程查险、险情抢护和洪水偎堤、漫滩情况查看时使用需要。

2. 对发生的黄河防洪工程险情，首先动用所在县级黄河河务局储备的防汛物资组织抢护，必要时按照程序申请紧急补充或从异地调用。沿黄各乡（镇）、村群众做好为抗洪抢险运送柳秸料的准备工作，接指令后按调用数量、时限要求保证供应到位。

3. 在各级政府统一组织领导下，充分利用社会储备和群众自备的救生器材和物资，投入滩区群众迁安救护，并做好受灾群众的安置工作。

4. 市、县级黄河河务部门，负责做好向同级或上级黄河防办报告物资储备、消耗及抢险设备动用情况的信息反馈工作，做好本辖区内物资调剂使用与补充库存工作，做到互通有无、相互支持。

（三）当花园口站发生6000～10000立方米每秒流量级洪水

我市黄河防汛将进入紧急状态。堤防、险工随时有出险的可能，渠村、刘楼堤段有可能发生顺堤行洪。此时物资保障重点是堤防险工、险段、涵闸等，保障方案为：

1. 黄河专业机动抢险队进入战备状态，全力做好参加抗洪抢险的准备工作。动用各县级黄河河务局储备的防汛冲锋舟保证抗洪指挥、工程查险、险情抢护和洪水漫滩、偎堤时使用。

2. 在各级防指的统一领导和组织下，动用黄河河务部门储备的防汛物资，保证其堤防险工、险段及涵闸工程巡查与抢护应急使用。沿黄各县、乡（镇）政府做好抗洪抢险用柳秸料的筹集和运送工作，接指令后按调用数量、时限要求保证供应到位。 

3. 社会各防汛物资储备单位，做好调用防汛储备物资的各项准备工作，按本级防指安排，调度时保证供应到位；有工程机械的社会各企、事业单位，做好设备投入抗洪抢险的准备工作，按防指调令及时到位投入抢险；各抗洪救灾物资生产企业和防汛物资储备单位，要按照防指命令，抓紧组织生产，积极筹措货源，按抗洪抢险需要保证供应；市、县交通运输部门要依照防汛职责，认真做好紧急抢运抗洪抢险物资的准备工作，保障物资运输及时到位。

4. 沿黄各县级黄河河务局按市黄河防办的统一安排，认真组织好国家储备防汛物资的调剂、调度和补充库存工作。濮阳黄河河务局负责向省防指黄河防办申请跨市区调剂、调度国家储备防汛物资，并根据省防指黄河防办的安排，做好国家储备防汛物资的调剂、调度和补充库存工作。物资部门做好工程抢险物资消耗、动用设备器材情况及主要防汛物资补充库存情况的信息统计与上报工作。

5. 在各级政府统一领导下，充分利用社会团体储备和群众自备的救生器材、漂浮工具、生活用品等救灾物资，开展滩区群众迁安救护工作，安置好受灾群众。

（四）当花园口站发生10000～15000立方米每秒流量级洪水时，我市黄河抗洪抢险斗争进入紧张阶段。滩区全部被淹，堤防工程全部偎水，堤防险点、隐患将暴露出来，所有坝岸工程随时有出大险的可能，巡堤查险、堤防和涵闸工程险情抢护及群众迁安救护任务十分繁重。此时物资保障重点是堤防全线，险工工程，涵闸工程和堤防防洪坝。保障方案为：

1. 动用各县级黄河河务部门储备的防汛物资，加强对堤防、险工、险段及涵闸（虹吸）等穿堤建筑物的查险、抢护和洪水下泄情况的查看等工作，确保及时掌握水情、工情、险情信息，确保工程安全。沿黄各县、乡（镇）政府要认真落实好抗洪抢险用柳秸料的筹集和供应工作，确保抢险需要。

2. 按照防指发布的汛情、险情、灾情信息和物资供应与调度命令，各级政府紧急动员，认真组织好各类防汛物资的筹集、供应和运输到位工作。社会各物资储备单位，要加强值班，明确责任，严明纪律，确保联络畅通。各级公安、交通运输部门要确保道路畅通，必要时采取交通管制，保证各类抗洪抢险物资安全及时运到抢险现场，满足抗洪需要。

3. 在各级政府的统一领导下，动用社会一切可利用的救生器材、漂浮工具和救灾物资，全力做好滩区群众迁安救护工作，尽最大努力减小损失。加强对黄河堤防工程险情的抢护，确保大堤安全。同时要加大对堤防工程、涵闸（虹吸）工程、顺堤行洪段的查险力度，确保洪水安全下泄。各防汛物资储备单位要认真做好防汛物资的调剂、调度和补充库存等工作，确保满足抗洪抢险物资需求。黄河专业机动抢险队、水上抢险队要按照省黄河防办下达的调度命令，积极投入抗洪抢险。

4. 充分利用国家常备资源，广泛调动社会各方资源，建立应急状态下物资征集调用工作机制，整合物资应急保障资源，实现物资优化配置和高效利用，保证应急状态下物资供应快速、高效。

5. 按照各级黄河防办要求，认真做好抗洪抢险防汛物资消耗、设备器材使用情况和主要防汛物资补充库存情况的统计、上报工作。

（五）当花园口站发生15000～22000立方米每秒流量级洪水时，我市黄河抗洪斗争形势十分严峻。堤防工程全部偎水，偎堤水深最深达6.2米，偎水越深堤段受到滚河、顺堤行洪的威胁越大，重大险情随时可能发生。堤防、涵闸（虹吸）工程防守任务十分繁重。此时物资保障重点是堤防，要将堤防抢险作为物资保障工作的重中之重，集中保障。保障方案为：

1. 在各级政府的统一指挥下，认真组织好人力、物力，严防死守，确保大堤安全。黄河专业机动抢险队、水上抢险队和当地所有能为抢险使用的设备，要调集到易发生险情的堤段，投入抗洪抢险。

2. 市、县、乡政府进行全民动员，组织落实好各类抢险救灾物资的供应。各抢险救灾物资生产厂家和防汛储备物资单位要按防指下达的命令，抓紧组织生产，积极筹措货源，保证抢险急需调用。市、县防指可根据汛情及物资可供资源状况，对生产货源实施封闭管制，确保统一调用。公安、交通运输部门要认真做好抢运抗洪救灾物资的各项工作，确保道路畅通，物资及时运输到位。

3. 在市、县政府的统一领导和指挥下，认真做好灾区群众的生活保障工作。各医疗、卫生、粮食、防疫、民政等部门要组织救灾人员、救灾物资，深入灾区为群众排忧解难。要妥善安排受灾群众特别是因灾转移群众的生活，使他们有饭吃、有衣穿、有住处、有病能得到及时医治。

4. 当花园口站洪水接近22000立方米每秒时，根据省防指命令，市政府、有关部队做好启用北金堤滞洪区实施分洪的相关准备工作，一旦命令下达，市政府、有关部队要按照指令要求组织落实到位。

（六）当花园口站发生22000立方米每秒以上超标准洪水时，我市抗洪救灾进入非常严峻时期，大险、恶险随时可能发生，堤防将面临严峻考验。有可能使用北金堤滞洪区滞洪。保障方案为：

1. 全市各级政府进行全民动员，组织社会上所有可能投入的人力、物力，抗御超标准洪水。尽最大努力满足抗洪需要求。由各类防汛专业队伍和人民解放军、民兵、企事业单位职工组成的抢险攻坚队伍，严防死守，全力抢护出险部位，尽最大努力确保大堤不决口，洪水安全下泄。

2. 按照市、县、乡政府的统一部署，落实好抗洪抢险物资生产、运输和调用，必要市防指统一协调，各县防指要严格按照指令落实抗洪抢险物资，确保调度申通，保障到位。

3. 市、县防指黄河防办要视情况预估抗洪物资需求，运筹可供资源，如需要动用中央储备，紧急向省防指黄河防办报告，请求给予支援。对接收到的物资要安排专人进行管理，保证不受损失，及时投入抗洪。

4. 渠村分洪闸闸前围堤按省防指命令做好爆破准备，等待国务院命令实施分洪。各单位储备的滞洪物资做好充分准备，一旦分洪，加强北金堤防守抢险，确保北金堤安全。

（七）各级洪水退落期间物资保障

1. 动用各县级黄河河务局储备的防汛冲锋舟，由所在地县级黄河河务局组织人员负责对黄河各类防洪工程查险、水毁工程查看，确保及时准确掌握水情、工情、险情，并将查看情况按规定上报。

2. 有承担国家储备、社会团体储备和群众备料黄河防汛物资储备任务的单位、部门、乡（镇）、村，仍须保持高度临战状态，做到险情不解除、思想不放松、人员不撤退；要按照各级黄河防办下达的物资调令，继续认真做好黄河抗洪抢险物资的供应与调度，确保工程抢护及险情恢复使用需要，夺取抗洪抢险的最后胜利。

我市黄河河务部门储备的黄河防汛物资只能用于黄河防洪工程抢险，未经省黄河防办批准，任何单位和个人不得挪用。动用时一律由市、县黄河防办具体办理并向上级黄河防办报告动用防汛物资、专用设备、器材的品种、数量。各级黄河防办在职权范围内研究决定动用本级防汛储备物资，并负责向物资储备单位下达“防汛物资调度通知”。

各县级黄河河务局储备的防汛冲锋舟，主要用于黄河抗洪指挥、工程查险、险情抢护和洪水漫滩、偎堤情况查看。物资储备单位在其管辖区域内且符合上述使用范围，由该类物资储备单位主管领导审批；超出上述使用范围，报上级防指黄河防办审批。

其他防汛物资、专用机械设备及防汛器材，在职权范围内，储备单位可根据抢险、查险使用情况报主管领导审批。

（二）动用社会团体储备和群众备料

动用社会团体储备和群众备料用于黄河抗洪抢险，由市防指及所在地县防指视情况安排，并报上级防指备案。物资储备单位、乡（镇）、村凭防指下达的物资调令供应到位。

（三）黄河专业机动抢险队

黄河专业机动抢险队（含水上抢险队）调度，由省、市防指黄河防办负责，未经省、市防指黄河防办批准，任何单位和个人不得动用。

一般情况下，黄河专业机动抢险队、水上抢险队负责市属区域内黄河防汛抢险，由市防指黄河防办负责调度。当花园口站发生4000立方米每秒以上流量级洪水时，黄河专业机动抢险队、水上抢险队由省防指黄河防办统一调度。

（四）上级黄河防办的代表在抢险工地现场指挥抢险时，紧急情况下，可临时批准动用防汛物资，事后应及时与本级黄河防办联系，报告处置结果并记录备案。

（五）在工程险情危急且抢险现场通讯联络中断时，抢险现场负责人可临时决定动用防汛物资，但必须在三日内向上级黄河防办书面报告，并按权限规定补办审批手续。

（六）动用防汛储备物资的请示和批复应正式行文，通过传真、电报传递。特殊情况下需电话请示或答复的，请示人和答复人必须将通话时间、通话内容、双方姓名、职务、工作部门记录备案，事后补办有关手续。

（七）部队和企、事业单位职工以及人民群众参加抗洪抢险，参加人员应自备、自带交通工具、通讯设备、抢险工器具及生活用品；抢险所需料物由抢险所在县黄河防办负责组织供应和调度。

（八）异地调用（从本市、县向外市、县调或从外市、县向本市、县调）防汛物资，按照“有偿使用、调物还钱（或调物还物）”、“谁申请、谁付款”的办法实施。调运过程中所发生的材料费、运杂费，由物资调出单位先行垫付，随后由物资申请调用单位结算。偿还物资过程中发生的购置费和运杂费均由物资申请调用单位负责。
